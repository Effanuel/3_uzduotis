# 3_uzduotis
Ši užduotis yra [antro](https://github.com/Effanuel/Duomenu-apdorojimas) darbo tęsinys

## Prerequisites
1. GNU C++ Compiler
2. *Optional: **gtest** for Unit Testing*

## Installation
1. [Atsisiūskite naujausią versiją.]()
2. Extract failus.
3. Per command-line:

```
cd 3_uzduotis
make all
project
```
